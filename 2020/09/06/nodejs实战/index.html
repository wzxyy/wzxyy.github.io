模拟英雄管理后台项目

写在开头

基于nodejs + express框架搭建后端项目的流程

  1. 初始化npm

    • npm init -y

  2. 安装模块

    • express
    • body-parser 接收post，保存post请求参数，在req.body中
    • fileUpload 接收post，保存post请求文件，在req.files中

  3. 使用模块

    • 创建服务器 let app = express();

    • 给req新增body属性

      • const bodyParser = require(‘body-parser’);
      • app.use(bodyParser.urlencoded({ extended: false }));

    • 接收formdata参数 （文件+文本） 给req增加files属性

      • const fileUpload = require(‘express-fileupload’);
