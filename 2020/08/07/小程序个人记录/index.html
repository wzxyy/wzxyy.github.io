登录小程序官网：https://mp.weixin.qq.com/

一、起步

1.注册账号

  • 注册界面：https://mp.weixin.qq.com/wxopen/waregister?action=step1

2.信息登记

  • 主体类型：个人

3.小程序信息填写

  • 服务类目：工具-信息查询

4.</>开发

  • 开发者ID：AppID

5.微信开发者工具

  • 下载安装：https://developers.weixin.qq.com/miniprogram/dev/devtools/devtools.html
  • 小程序开发文档：https://developers.weixin.qq.com/miniprogram/dev/framework/

6.熟悉开发者工具

  • 菜单栏

    • 新建项目
    • 导入项目，打开已有项目。在微信开发者工具里面，不能拖入打开
    • 打开最近的项目，会展示一个最近项目列表，点击即可打开
    • 设置-外观 可以设置深色模式
    • 设置->编辑

  • 工具栏

    • 编辑器，模拟器，调试器，点击是否显示对应模块，当然最少得显示一个

    • 编译：手动编译

    • 预览：点击出现二维码，手机扫码，就可以在手机微信里面查看效果

    • 自动预览，自动刷新手机上预览效果

      • 模拟器就是用来查看效果

        • 切换机型

        • 显示比例

  • 编辑器

    • 相当于是vscode, 快捷键和vscode完全一致
    • 左侧是项目结构，右侧是编辑文件
    • 有git功能，好用

  • 调试器

    • wxml是结构
    • console是打印
    • sources查看源码，方便打断点调试
    • network查看请求
    • storage存储
    • 终端和vscode一致

  • 模拟器

    • 看运行效果的

7.新建项目

7.1 小程序

  • 新手不要选云开发，先熟悉基础的开发模式哦

  • 需要选择一个空文件夹

  • 开发者工具AppId

    • 真实的，微信开发者工具所有的菜单栏都可以用
    • 测试号，微信开发者工具不能上传
    • appid格式填错了，微信开发者工具，不能预览也不能真机调试，也不能上传

7.2 项目结构

  • pages
    • 所有页面都存放在这里，而且这里只能存放页面
    • 文件名必须一致
      • .js 页面逻辑
      • .json 页面配置
      • .wxml 页面结构
      • .wxss 页面样式
  • app.js
    • 小程序入口文件
  • app.json
  • app.wxss 全局样式

二、基本标签的使用

1.准备工作

  • 快速创建页面结构
    • 在 app.json 的 pages 中直接写上新的路径，会自动生成文件

2.认识标签

2.1 text

2.2 view

2.3 image

2.4 button

  • 默认是独占一行的，如果需要并排，需要加size属性
    • size = “ mini ”

注意：小程序不认识的标签会被忽略掉，基本上和text一样，尽量不要使用非小程序组件的标签

三、数据绑定

小程序与 vue 一样，也可以用指令来操作 dom：https://developers.weixin.qq.com/miniprogram/dev/reference/wxml/data.html

3.1 插值语法

  • 数据绑定使用 双大括号 将变量包起来

四、事件

注册事件：https://developers.weixin.qq.com/miniprogram/dev/framework/view/wxml/event.html

语法：bind事件名 =”事件处理方法”

4.1 tap事件

  • 相当于 pc 端的 click，移动端叫触碰
  • 方法：fn
    • 在 .js 中编写，直接与 data 同级创建一个 fn 函数就行了

4.2 showToast消息提示

  • 小程序里面，大多数API是由”wx”对象提供的

  • wx 相当于vue的 this

五、this

5.1 方法中this

小程序中的 this 指向就的当前小程序页面实例，可以通过 this 取得对象上的属性与方法

  • 访问 data 属性

    • this.data.属性

  • 访问方法

    • this.方法名

  • 改变 data 属性

    • this.setData({ 属性名：值 })

    • 更改 data 中的属性，并在页面中同步需要用此方法

六、模块化

小程序中也支持模块化的概念

6.1作用

  • 把一些公共的代码抽离作为一个单独 js

6.2 使用

  • 和 ES6 的语法一样
  • 注意：在 import 中的不支持绝对路径
